# pbx
PBX in docker container based on Kamailio and Asterisk

## Getting Started
1. Obtain code
```
$ git clone https://github.com/genajan/pbx.git
```
2. Go to the project directory
```
$ cd pbx
```
3. Edit .env file. Set domain name and IP address.
4. Start container
```
$ docker-compose up -d
```
